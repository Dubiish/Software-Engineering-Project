# Software-Engineering-Project
Project for software engineering class

## Description
- Web application used by company which sells textile products
- As everything is getting digitized, even small companies/shop want to stay relative in this digital world
- Switching to web application make a lot of tasks and calculations much easier
- For example searching through orders and customers via filter on webstie is much faster than searching through dozen of papers
- Application should be used only by employees

## Technologies used
- Application can be divided into 2 parts
    - Front-end
    - Back-end
- For front-end -> **React.js** framework with CSS framework **Bootstrap**
- For back-end -> **Node.js** server with **Express.js** framework
- For data storage -> **MySQL** database

## Languages Used
- For React.js, Node.js and express.js -> **JavaScript/TypeScript** with some **HTML, CSS**
- For database -> **SQL**
